================================================================================
                    ELITE GYM WEBSITE - PAGES & FEATURES LIST
================================================================================

This document provides a complete list of all pages and their features in the
Elite Gym website.

================================================================================
                            FRONTEND PAGES (Public)
================================================================================

1. HOME PAGE (/)
   Features:
   - Fullscreen hero section with video background
   - Animated call-to-action buttons (Get Started, Learn More)
   - Statistics counter section with animated numbers:
     * 5000+ Active Members
     * 50+ Expert Trainers
     * 100+ Classes Weekly
     * 15+ Years Experience
   - Programs preview section (shows first 3 programs)
   - Expert Trainers showcase (shows first 3 trainers)
   - Membership Plans preview (shows first 3 plans)
   - Customer Testimonials section (3 testimonials)
   - Call-to-action section with gradient background
   - Scroll indicator animation
   - Smooth page transitions and animations

2. ABOUT PAGE (/about)
   Features:
   - Hero section with gym story introduction
   - Mission and Vision sections (side by side)
   - "Why Choose Us" features grid:
     * State-of-the-Art Equipment
     * Expert Trainers
     * Personalized Programs
     * Flexible Timings
     * Community Support
     * Progress Tracking
   - Our Story section with detailed narrative
   - Animated feature cards with hover effects

3. PROGRAMS PAGE (/programs)
   Features:
   - Display all available fitness programs
   - Program cards showing:
     * Program name and description
     * Duration and difficulty level
     * Features list
     * Pricing information
     * Program images
   - Responsive grid layout (1-3 columns)
   - Loading states
   - Error handling
   - "What's Included in All Plans" information section

4. TRAINERS PAGE (/trainers)
   Features:
   - Display all gym trainers
   - Trainer cards showing:
     * Trainer name and photo
     * Specialization and experience
     * Social media links (Instagram, Facebook, Twitter)
     * Bio information
   - Responsive grid layout
   - Social media icons (clickable links)

5. PLANS PAGE (/plans)
   Features:
   - Display all membership plans
   - Plan cards showing:
     * Plan name and price
     * Features and benefits list
     * Duration options
     * Razorpay payment integration button
   - Payment flow integration
   - Demo mode support
   - "What's Included in All Plans" section

6. TIMETABLE PAGE (/timetable)
   Features:
   - Weekly class schedule display
   - Day selector (Monday through Sunday)
   - Class information showing:
     * Class name
     * Trainer name
     * Time and duration
     * Capacity information
   - Filter classes by day
   - Responsive layout

7. GALLERY PAGE (/gallery)
   Features:
   - Image gallery grid layout
   - Lightbox functionality for viewing images in full size
   - Responsive image grid
   - Navigation between images in lightbox

8. TESTIMONIALS PAGE (/testimonials)
   Features:
   - Display customer testimonials
   - Testimonial cards with:
     * Customer name and photo
     * Rating stars (1-5 stars)
     * Review text
   - Grid layout
   - Smooth animations on scroll

9. TRANSFORMATIONS PAGE (/transformations)
   Features:
   - Gallery of member transformation stories
   - Before/after photos
   - Member testimonials and success stories
   - Transformation cards with images and stories
   - Approved transformations only (admin-reviewed)

10. CONTACT PAGE (/contact)
    Features:
    - Contact form with fields:
      * Name (required)
      * Email (required)
      * Phone (optional)
      * Subject (required)
      * Message (required)
    - Contact information display:
      * Gym address
      * Phone numbers
      * Email addresses
      * Operating hours
    - Form validation
    - Success/error messages
    - Contact submissions stored in database

11. TOOLS PAGE (/tools)
    Features:
    - BMI Calculator:
      * Height and weight input sliders
      * Real-time BMI calculation
      * BMI category display (Underweight/Healthy/Overweight/Obese)
      * Color-coded status badges
    - Body Fat Calculator:
      * Navy formula body fat estimation
      * Gender-specific calculations (Male/Female)
      * Waist, neck, and hip measurements
      * Body fat percentage display
      * Fitness range indicators
    - TDEE Calculator (Total Daily Energy Expenditure):
      * Age, height, weight inputs
      * Activity level selection (Sedentary to Athlete)
      * Maintenance calories calculation
      * Cutting and bulking calorie targets
      * Based on Mifflin-St Jeor equation
    - Workout and Diet Plan Previews:
      * Shows preview of available workout plans
      * Shows preview of available diet plans
      * Links to full access for members
    - Book Assessment call-to-action

12. EXERCISES PAGE (/exercises)
    Features:
    - Comprehensive exercise library
    - Search functionality by exercise name
    - Advanced filtering system:
      * Muscle Group (Chest, Back, Shoulders, Arms, Legs, Core)
      * Sub-Muscles (within each muscle group)
      * Equipment type
      * Difficulty level (Beginner, Intermediate, Advanced)
      * Exercise Goal
      * Movement Pattern (Push, Pull, Squat, Hinge, etc.)
      * Exercise Type
    - Exercise cards showing:
      * Exercise name and image
      * Description
      * Difficulty badge
      * Muscle group tags
      * Movement type tags
      * Equipment requirements
    - Filter panel (collapsible)
    - Active filters display
    - Clear all filters button
    - Responsive grid layout

13. EXERCISE DETAIL PAGE (/exercises/:id)
    Features:
    - Detailed exercise information
    - Exercise images/videos
    - Instructions and form tips
    - Muscle groups targeted
    - Equipment needed
    - Difficulty level
    - Related exercises

14. HELP CENTER PAGE (/help-center)
    Features:
    - Frequently Asked Questions display
    - Search functionality for questions
    - Categorized FAQ sections
    - Answers to common queries
    - Real-time search filtering

15. LOGIN PAGE (/login)
    Features:
    - User authentication form
    - Email and password login
    - Registration option for new users
    - "Remember me" functionality
    - Error messages for invalid credentials
    - Redirect to dashboard after login

================================================================================
                    USER DASHBOARD PAGES (Protected)
================================================================================

16. DASHBOARD OVERVIEW (/dashboard)
    Features:
    - Personalized welcome message
    - Account status display (Member/Visitor)
    - Membership status and expiry information
    - Quick stats cards:
      * Account Status
      * Membership Plan
      * Days Remaining
    - Tabbed interface:
      * Overview tab
      * Membership tab
      * Profile tab
    - Quick Actions cards:
      * Check-In (Attendance)
      * Workout Library
      * Nutrition Hub
      * Progress Tracker
      * Transformation Submission
    - Membership purchase options (for visitors)
    - Membership details display (for members)
    - Membership expiry countdown
    - Renewal alerts (7 days before expiry)
    - Profile information display
    - Payment integration for plan purchases

17. DASHBOARD WORKOUTS (/dashboard/workouts)
    Features:
    - View assigned workout plans
    - Browse all available workout plans
    - Create custom workout plans
    - Edit user-created workout plans
    - Delete user-created workout plans
    - Add exercises to custom plans
    - Exercise details and instructions
    - Video demonstrations (if available)
    - Progress tracking for workouts
    - Mark workouts as completed
    - Filter workouts by:
      * Assigned vs All
      * User-created vs System plans
    - Search functionality
    - Exercise library integration
    - Workout plan sharing

18. DASHBOARD DIET (/dashboard/diet)
    Features:
    - View assigned diet plans
    - Browse all available diet plans
    - Meal plans with recipes
    - Macro and calorie information
    - Daily meal schedules
    - Shopping lists
    - Filter by goals (weight loss, muscle gain, maintenance)
    - Nutritional breakdown display

19. DASHBOARD PROGRESS (/dashboard/progress)
    Features:
    - Log body measurements (weight, body fat, measurements)
    - Track progress over time
    - Visual charts and graphs
    - Goal setting
    - Progress photos upload
    - Historical data view
    - Date-based progress entries
    - Measurement tracking (chest, waist, arms, etc.)

20. DASHBOARD ATTENDANCE (/dashboard/attendance)
    Features:
    - Check-in functionality
    - Attendance history
    - Attendance statistics
    - Date-based attendance tracking
    - Attendance calendar view

21. TRANSFORMATION SUBMIT (/dashboard/transformation-submit)
    Features:
    - Submit before/after photos
    - Write transformation story
    - Upload multiple photos
    - Status tracking (pending/approved/rejected)
    - Time period of transformation
    - Form validation
